# Swiggy Web Traffic & Conversion Analysis

## 📌 Project Overview
This project analyzes Swiggy’s website traffic to understand user behavior, traffic quality, and session-level conversion performance. The objective is to identify opportunities to improve engagement and conversions using data-driven insights.

The analysis is presented through an interactive Tableau dashboard designed for marketing and product stakeholders.

---

## 🎯 Business Objective
- Evaluate web traffic quality across acquisition channels
- Measure session-level conversion efficiency
- Identify engagement issues using bounce rate and time-on-site metrics
- Enable stakeholder exploration using interactive filters

---

## 📊 Key Metrics
- Total Website Sessions
- Session Conversion Rate
- Session Bounce Rate
- Average Time on Site
- Daily Sessions Trend
- Daily Conversion Rate Trend

---

## 🧠 Key Insights
- Conversion rates are consistent across channels and devices (~40–41%)
- Bounce rates vary more significantly, indicating traffic quality or early UX issues
- Mobile users show slightly higher bounce rates but similar conversion performance
- Performance trends are stable with occasional campaign or seasonal dips
